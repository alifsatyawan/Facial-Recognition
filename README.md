# Facial-Recognition
📹 Built a facial recognition algorithm from scratch using computer vision by leveraging OpenCV to process images and NumPy for numerical operations.

Achieved a confidence level of 90% by experimenting with a diverse set of images sourced from a repository.
Optimized the recognition process to identify multiple individuals in a single image, handling up to 50 faces
concurrently using the face_recognition library and Python scripting.
